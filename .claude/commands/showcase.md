# /sigil showcase

Launch the visual component browser for exploring captured taste.

## Usage

```
/sigil showcase
/sigil showcase --build
/sigil showcase --init
```

## Options

| Option | Description |
|--------|-------------|
| `--build` | Build for production instead of dev server |
| `--init` | Initialize showcase app from template |
| `--port N` | Use specific port (default: 3000) |

## Prerequisites

- Node.js 18+ installed
- Sigil mounted (`/sigil mount`)
- Components exported (`/sigil export`)

## First Time Setup

If showcase app doesn't exist, initialize it:

```bash
/sigil showcase --init
```

This copies the template from `~/.sigil/templates/showcase-app/` to `sigil-showcase/showcase/`.

## Workflow

```
/sigil taste              # Capture component taste
/sigil export             # Generate components.json
/sigil showcase           # Browse in browser
```

## Features

### Component Browser

- Filter by tier (Gold, Silver, All)
- Filter by JTBD intent
- Search by component name
- View tier statistics

### Component Detail

- Full taste profile display
- Rejected patterns list
- Inspiration references
- Source file location

### Live Playground

- State switcher (default, hover, loading, success, error)
- Physics visualization (Gold only)
- Spring parameter display
- Interactive preview

## Directory Structure

```
sigil-showcase/
├── exports/
│   └── components.json     # Generated by /sigil export
└── showcase/
    ├── app/                # Next.js app
    ├── components/         # UI components
    ├── lib/                # Registry and types
    └── public/
        └── components.json # Symlink to exports
```

## Scripts

The showcase app includes:

| Script | Description |
|--------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Build for production |
| `pnpm export` | Sync component data |

## Error Handling

| Condition | Message |
|-----------|---------|
| Not mounted | "Sigil not mounted. Run /sigil mount first." |
| No exports | "No exports found. Run /sigil export first." |
| Node missing | "Node.js 18+ required. Install from nodejs.org" |
| Port in use | "Port 3000 in use. Try --port 3001" |

## Customization

Edit `sigil-showcase/showcase/` to customize:
- `tailwind.config.ts` - Colors and theme
- `app/layout.tsx` - Navigation and branding
- `components/` - UI components
